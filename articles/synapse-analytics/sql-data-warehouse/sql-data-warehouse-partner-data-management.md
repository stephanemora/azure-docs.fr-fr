# <a name="azure-synapse-analytics-data-management-partners"></a>Partenaires de gestion de données d’Azure Synapse Analytics

Pour créer une solution d’entrepôt de données utilisant la fonctionnalité de pool SQL Synapse d’Azure Synapse, vous disposez d’une vaste panoplie d’outils de pointe. Cet article met en évidence les sociétés partenaires de Microsoft qui proposent des outils et solutions de gestion de données prenant en charge Azure Synapse Analytics.
