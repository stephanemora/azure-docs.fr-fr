# <a name="about-your-application"></a>À propos de votre application

Cet article vous montre comment obtenir des informations sur votre application IoT Central. Vous aurez besoin de :

- Ces informations si vous contactez le support technique.
- L’abonnement Azure que votre application utilise pour localiser les informations de facturation dans le Portail Azure.
- L’ID de l’application lorsque vous utilisez l’API REST.
- La version de l’application pour effectuer des tâches telles que l’ajout d’un connecteur.

## <a name="get-information-about-your-application"></a>Obtenir des informations sur votre application

Obtenir des informations sur votre application IoT Central :

1. Sélectionnez le lien **Aide** dans le menu du haut.

1. Sélectionnez **À propos de votre application**.

1. La page **À propos de votre application** affiche des informations sur votre application :  

    :::image type="content" source="media/howto-get-app-info/about-your-app.png" alt-text="À propos de la capture d’écran de votre application":::

    Utilisez le bouton **Copier les informations** pour copier les informations de la grille dans le presse-papiers.

## <a name="next-steps"></a>Étapes suivantes

Maintenant que vous savez comment trouver la version de votre application IoT Central, l’étape suivante consiste à continuer à explorer les articles de procédure pour les administrateurs : [Modifier les paramètres d’application IoT Central](howto-administer.md).

Si vous avez une application v2, consultez [Migrer votre application IoT Central v2 vers v3](howto-migrate.md).
