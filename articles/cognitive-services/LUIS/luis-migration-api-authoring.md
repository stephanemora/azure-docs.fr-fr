# <a name="authoring-api-changes"></a>Modifications apportées aux API de création 

Les API de création sont mises à jour pour V3 afin de permettre la [décomposition de modèles avec des entités apprises automatiquement](luis-concept-data-extraction.md). 

Vous pouvez créer dans les API V2 ou V3 en préversion. La même application peut également choisir d’utiliser des réponses de point de terminaison de prédiction V2 ou V3. Vous n’avez pas besoin de disposer de la même version d’API pour la création et la réponse de point de terminaison de prédiction.

Le portail LUIS utilise exclusivement les API de création V3.

## <a name="next-steps"></a>Étapes suivantes

Passez en revue les [modifications apportées au point de terminaison de prédiction](luis-migration-api-v3.md) pour l’API V3.
