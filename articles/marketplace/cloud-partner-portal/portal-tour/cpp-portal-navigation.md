# <a name="cloud-partner-portal-navigation"></a>Navigation dans le portail Microsoft Cloud Partner

La navigation dans le [portail Microsoft Cloud Partner](https://cloudpartner.azure.com) est possible au moyen de deux principaux mécanismes d’interface utilisateur : la barre de menus du haut et le volet de navigation de gauche.  Ces éléments d’interface utilisateur restent identiques et disponibles quelle que soit la page du portail à laquelle vous accédez.


## <a name="menu-bar"></a>Barre de menus

La barre de menus s’affiche en haut du site du portail.  Elle comporte les quatre zones actives suivantes :

![Barre de menus](./media/top-menubar1.png)

Un clic sur le contenu de ces zones déclenche les actions suivantes :

1. Zone de titre : ouvre la page d’accueil du portail.
2. Bouton **Aide** : affiche le menu Aide, qui fournit des ressources utiles pour l’utilisateur.
3. Bouton **Commentaires** : affiche le formulaire **Envoyez-nous vos commentaires**.
4. Bouton **Éditeur** : affiche un menu qui permet à l’utilisateur de gérer son profil.

Ces menus et formulaires sont décrits ci-après.

### <a name="help-menu"></a>Menu aide

Le menu **Aide** fournit des ressources utiles pour l’utilisateur du portail Microsoft Cloud Partner.

![Barre de menus du haut](./media/top-menubar2.png)

Le menu Aide contient les éléments suivants :

- **Documentation** : ouvre un nouvel onglet dans le navigateur et affiche la page d’accueil de la documentation relative au portail Microsoft Cloud Partner. 
- **Forum** : ouvre un nouvel onglet et affiche la page d’accueil de la documentation de la Communauté de partenaires pour les Places de marché AppSource et Azure.
- **Support** : ouvre un nouvel onglet et affiche la page de support pour la publication sur la Place de marché, où vous pouvez créer un rapport d’incident.  Nous trions et passons en revue tous les commentaires dans le respect constant de votre vie privée. Nous vous contactons uniquement si vous en faites la demande ou si vous avez besoin d’informations complémentaires.


### <a name="send-us-feedback-form"></a>Envoyez-nous vos commentaires

Le formulaire **Envoyez-nous vos commentaires** vous permet d’envoyer vos commentaires directement à l’équipe d’intégration des éditeurs de la Place de marché.  Nous vous contactons uniquement pour obtenir des informations supplémentaires si vous avez coché la case **OK pour vous contacter au sujet de vos commentaires**.

![Formulaire de commentaires](./media/feedback-form.png)

> [!TIP]
> Si vos commentaires portent sur une page spécifique du portail Microsoft Cloud Partner, accédez d’abord à cette page avant de cliquer sur le bouton **Commentaires**.  Vérifiez que la case **Inclure une capture d’écran** est cochée avant d’envoyer vos commentaires. Cela garantit que l’état de l’interface utilisateur de votre session pourra être capturé. 


### <a name="publisher-menu"></a>Menu Éditeur

Ce menu vous permet de gérer votre profil et votre session, notamment accéder directement à la page de votre profil d’éditeur, afficher le contrat d’éditeur Microsoft ou vous déconnecter de la session active. 

![Menu Éditeur](./media/publisher-menu.png)


## <a name="left-navigation-pane"></a>Volet de navigation de gauche

Un volet de navigation réductible occupe le côté gauche du portail.  Il contient une barre de menus dont les éléments suivants permettent aux partenaires d’effectuer différentes actions :


|    **Élément de menu**     |      **Description**                       |
|    -------------     |      ---------------                       |
| ![Élément Nouvelle offre](./media/left-navbar1.png) | Le [menu Nouvelle offre](./cpp-new-offer-menu.md) permet de créer des offres du type spécifié. |
| ![Élément Toutes les offres](./media/left-navbar2.png) | La page [Toutes les offres](./cpp-all-offers-page.md) permet de voir toutes les offres publiées ainsi que certaines de leurs caractéristiques principales. |
| ![Élément Approbations](./media/left-navbar3.png) | La page [Approbations](./cpp-approvals-page.md) permet de voir toutes leurs offres actuellement approuvées. |
| ![Élément Tous les éditeurs](./media/left-navbar4.png) | La page [Tous les éditeurs](./cpp-all-publishers-page.md) liste tous les éditeurs actuellement inscrits dans Cloud Partner. |
| ![Élément Utilisateurs](./media/left-navbar5.png) | La page [Utilisateurs](./cpp-users-page.md) liste tous les comptes d’utilisateur actuellement associés à des partenaires inscrits dans Cloud Partner. |
| ![Élément Historique](./media/left-navbar6.png) | La page [Historique](./cpp-history-page.md) liste les événements de publication et de modification d’offres pour tous les éditeurs. |
| ![Élément Insights](./media/left-navbar7.png) | La page [Insights](./cpp-insights-page.md) contient les différents tableaux de bord qui composent la fonctionnalité *Seller Insights* de ce portail. |
| ![Élément Documentation](./media/left-navbar8.png) | Ouvre un nouvel onglet et affiche la page d’accueil de la documentation relative au portail Microsoft Cloud Partner (entrée identique dans le menu **Aide**). |
| ![Élément Forum](./media/left-navbar9.png)  | Ouvre un nouvel onglet et affiche la page d’accueil de la documentation de la Communauté de partenaires pour les Places de marché AppSource et Azure (entrée identique dans le menu **Aide**). |
|  |  |

> [!NOTE]
> Comme cette barre de menus est régulièrement mise à jour pour refléter les nouvelles fonctionnalités du portail, les éléments que vous voyez peuvent différer de ceux listés dans le tableau ci-dessus.
