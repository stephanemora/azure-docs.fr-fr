# <a name="azure-monitor-autoscale-actions-resource-log-schema"></a>Schéma du journal des ressources des actions de mise à l’échelle automatique Azure Monitor

Voici les formats généraux pour les journaux de ressources de mise à l’échelle automatique avec des exemples de données inclus. Tous les exemples ci-dessous ne sont pas au format JSON correctement formé, car ils peuvent inclure plusieurs valeurs valides pour un champ donné. 

Utilisez les événements de ce type pour résoudre les problèmes que vous pouvez rencontrer avec la mise à l’échelle automatique. Pour plus d’informations, consultez [Dépannage des problèmes liés à la mise à l’échelle automatique](autoscale-troubleshoot.md).


## <a name="profile-evaluation"></a>Évaluation du profil

Enregistrement lorsque la mise à l’échelle automatique consulte pour la première fois un profil de mise à l’échelle automatique

```json
{
  "time": "2018-09-10 18:12:00.6132593",
  "resourceId": "/SUBSCRIPTIONS/BA13C41D-C957-4774-8A37-092D62ACFC85/RESOURCEGROUPS/AUTOSCALETRACKING12042017/PROVIDERS/MICROSOFT.INSIGHTS/AUTOSCALESETTINGS/DEFAULTSETTING",
  "operationName": ["FixedDateProfileEvaluation", "RecurrentProfileEvaluation", "DefaultProfileEvaluation"],
  "category": "AutoscaleEvaluations",
  "correlationId": "e8f67045-f381-445d-bc2d-eeff81ec0d77",
  "property": {
    "targetResourceId": "/subscriptions/d45c994a-809b-4cb3-a952-e75f8c488d23/resourceGroups/RingAhoy/providers/Microsoft.Web/serverfarms/ringahoy",
    "profile": "defaultProfile",
    "profileSelected": [true, false]
  }
}
```

## <a name="profile-cooldown-evaluation"></a>Évaluation du temps de recharge (cooldown) du profil

Enregistrement lorsque la mise à l’échelle automatique évalue si elle ne doit pas exécuter une mise à l’échelle en raison d’une période de recharge (cooldown). 

```json
{
  "time": "2018-09-10 18:12:00.6132593",
  "resourceId": "/SUBSCRIPTIONS/BA13C41D-C957-4774-8A37-092D62ACFC85/RESOURCEGROUPS/AUTOSCALETRACKING12042017/PROVIDERS/MICROSOFT.INSIGHTS/AUTOSCALESETTINGS/DEFAULTSETTING",
  "operationName": "ScaleRuleCooldownEvaluation",
  "category": "AutoscaleEvaluations",
  "correlationId": "e8f67045-f381-445d-bc2d-eeff81ec0d77",
  "property": {
    "targetResourceId": "/subscriptions/d45c994a-809b-4cb3-a952-e75f8c488d23/resourceGroups/RingAhoy/providers/Microsoft.Web/serverfarms/ringahoy",
    "selectedProfile": "defaultProfile",
    "scaleDirection": ["Increase", "Decrease"]
    "lastScaleActionTime": "2018-09-10 18:08:00.6132593",
    "cooldown": "00:30:00",
    "evaluationTime": "2018-09-10 18:11:00.6132593",
    "skipRuleEvaluationForCooldown": true
  }
}
```
