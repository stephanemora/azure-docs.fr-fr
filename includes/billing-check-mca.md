Vérifiez le type de contrat pour déterminer si vous avez accès à un compte de facturation d’un contrat de client de Microsoft.

1. Connectez-vous au [Portail Azure]( http://portal.azure.com).

2. Effectuez une recherche sur **Gestion des coûts + facturation**.

   ![Capture d’écran montrant la recherche dans le portail Azure](./media/billing-check-mca/billing-search-cost-management-billing.png)

3. Si vous avez accès à un seul compte de facturation, sélectionnez **propriétés** à partir de la partie gauche. Vous avez accès à un compte de facturation d’un contrat de client de Microsoft si le type de compte de facturation est **contrat de client Microsoft**.

    ![Capture d’écran montrant un contrat de client de microsoft dans la page de propriétés](./media/billing-check-mca/billing-mca-property.png)

4. Si vous avez accès à plusieurs comptes de facturation, vérifiez le type de la colonne de compte de facturation. Vous avez accès à un compte de facturation d’un contrat de client de Microsoft si le type pour les comptes de facturation est **contrat de client Microsoft**.

    ![Capture d’écran montrant un contrat de client de microsoft dans la page de liste de compte facturation](./media/billing-check-mca/billing-mca-in-the-list.png)
