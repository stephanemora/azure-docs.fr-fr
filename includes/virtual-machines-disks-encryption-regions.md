Pour les disques SSD Premium, les disques SSD standard et les disques durs standard, seules les régions suivantes prennent actuellement en charge les clés gérées par le client :

- Disponible en tant qu’offre de disponibilité générale dans les régions publiques et les régions Azure Government.

Pour les disques Ultra, seules les régions suivantes prennent en charge les clés gérées par le client en tant qu’offre de disponibilité générale :

- USA Est
- USA Ouest 2
- États-Unis - partie centrale méridionale 
- Gouvernement américain - Virginie
